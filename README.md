# Moringa AI Capstone - Solidity Zombie Factory

A professional-grade Beginner's Toolkit for Solidity development using Hardhat. This project implements a "Zombie Factory" inspired by the CryptoZombies curriculum.

## Features
- **Smart Contract:** `ZombieFactory.sol` with structs, events, and DNA generation.
- **Testing Suite:** Comprehensive Hardhat tests to verify contract logic and event emission.
- **Development Environment:** Full Hardhat integration for compiling and testing.

## Prerequisites
- **Node.js** (v16+)
- **NPM**

## Getting Started
1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Compile the smart contract:
   ```bash
   npx hardhat compile
   ```
4. Run the unit tests:
   ```bash
   npx hardhat test
   ```

## Project Structure
- `contracts/`: Contains the Solidity source files.
- `test/`: Contains the automated test suite.
- `hardhat.config.js`: Hardhat configuration.
- `Toolkit Document.md`: Detailed beginner's guide and AI learning journal.
